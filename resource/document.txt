c_mail document:

    http://domain/libs/c_email/send

        必须参数email_subject, email_to, email_message 
        示例../send?email_to=example@mail.com&email_subject=test email&email_message=helloworld
        {   用于发送一封邮件    }


    http://domain/libs/c_email/validate/$1

        必须在url中传入publickey值用来验证 publickey是否存在
        示例../validate/[publickey]
        {   用于验证注册邮件在中的publickey    }


    http://domain/libs/c_email/send_reg

        必须参数email_to, keyword
        示例../send_reg?email_to=example@mail.com&keyword=php
        {   用于发送一封邮箱验证的邮件   }


    http://domain/libs/c_email/send_all
        必须参数email_subject, email_message
        示例../send_all?email_subject=test email&email_message=hellworld
        {   用于对已注册并验证通过的用户遍历发送邮件    }


    http://domain/libs/c_email/add_data
        必须参数email_to, keyword, location
        示例.../libs/c_email/add_data?email_to=example@mail.com&keyword=java&location=bejing
        {   对于已注册用户增加关键字以及地点    }


    http://domain/libs/c_email/send_all_push
        无参
        示例.../libs/c_email/send_all_push
        {   对已注册用户并已订阅用户发送职位内容邮件，默认发送10条内容。邮件主模版路径[ resource/template_1.html ]， 组模版路径[ resource/template_1_1.html ]   }


    http://domain/libs/c_email/subscribe
        必须参数email_to value
        示例.../libs/c_email/subscribe?email_to&example@mail.com&value=1 [ value: 1(取消订阅) or 0(订阅)  ]
        {  对用户订阅状态进行控制  }


    http://domain/libs/asyn?pass=***
        必须参数：pass = ***
        示例.../libs/asyn?pass=***	[ pass验证信息，否则需要用户登录 ]
        {  增加异步发送邮件，解决SMTP服务器缓慢发送邮件时对客户端数据请求堵塞问题  }    


tip:
    用户定位及关键字在数据库中的数据结构
        JSON : {"php":["xian","xii"],"java":["xii","ankang"]}
        PHP : array(keyword, array(location, ..., ...),  keyword array(location, ..., ...), ...)
